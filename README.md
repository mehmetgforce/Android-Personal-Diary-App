# Android-Personal-Diary-App
Android Studio ile geliştirilen çoklu ekran (Activity) yapısına sahip kişisel günlük uygulamasıdır. Kullanıcıdan alınan metin verilerinin Intent yapıları ile diğer sayfaya aktarılmasını ve UI (XML) tasarım temellerini içerir. Yazılım mühendisliği mobil geliştirme pratikleri kapsamında tasarlanmış temel bir Android projesidir.
